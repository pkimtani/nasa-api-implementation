## Description

NestJS based implementation for NASA API to show near earth objects.

## Installation

```bash
$ npm install
```

## Running the app

Pre-requisite:
`cp .env.example .env` and enter your NASA API key.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Head to `localhost:3000/api` to interact and use the API.

## Test

```bash
# unit tests
$ npm run test
```
